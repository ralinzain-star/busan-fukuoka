8Time-aware Features Dynamic & Contextual

網站會根據「現在是什麼時候」自動調整顯示內容，從出發前的待辦清單到旅行中的即時行程進度。

Today Overlay — 全螢幕即時助手 Dynamic
> 7 天前：確認機票、飯店、票券、行程
≤ 7 天：確認票券、飯店、購買 eSIM、查天氣
≤ 3 天：打包行李、eSIM、換現金、確認票券
≤ 1 天：打包、護照、eSIM、換現金、充電器、線上報到連結
Countdown Timer — 秒級三態切換
setInterval(1000) 每秒刷新。出發前倒數至 IT606 航班 16:40 TPE（+08:00），旅行中倒數至 4/12 23:59 JST（+09:00），結束後歸零。
Today Itinerary Timeline
旅行中事件時間線：已過變淡 .past、進行中高亮 .current。每 60 秒刷新。
Entry Doc Reminders
偵測未完成的 Visit Japan Web / K-ETA，自動顯示在 overlay。
Inline Weather
旅行中 overlay 嵌入當天天氣：溫度、描述、日出日落。
Scroll Lock
Overlay 開啟時鎖定背景捲動，關閉後恢復。
